🎮 Simon Game Instructions

1. Press any key to start the game.

2. Watch carefully as a colored button flashes and plays a sound.

3. Click the buttons in the exact order shown by the game.

4. Each level adds a new color to the sequence.

5. Repeat the entire sequence correctly to advance.

6. If you click the wrong button, the game is over.

7. Press any key after "Game Over" to start a new game.
